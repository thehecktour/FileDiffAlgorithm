# Pokemon Password Generator API

#### Este projeto é uma API construída com NestJS que gera senhas aleatórias baseadas em nomes de Pokémon, utilizando a API PokeAPI. A aplicação faz uma chamada à API PokeAPI para obter nomes de Pokémon e usa esses nomes para criar senhas únicas.

### Instalação

Clone o repositório:

bash
``````
  git clone https://github.com/seu-usuario/PasswordGenerator.git
  cd PasswordGenerator
``````

### Instale as dependências:

##### Certifique-se de ter o Node.js instalado. Em seguida, execute:

bash
``````
  npm install
``````

### Inicie o servidor:

bash
``````
  npm run start
``````
##### O servidor estará rodando em http://localhost:3000.

### Uso

##### Para usar a API, faça uma requisição HTTP para o endpoint /user/password. A API retornará uma senha aleatória baseada no nome de um Pokémon.
